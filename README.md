# AI Disaster Assistant

An AI-powered web application that provides real-time survival guidance during disasters.

## Features
- AI-generated emergency responses
- Location-based suggestions
- Multi-language support (planned)
- Panic mode for instant help

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- AI: OpenAI API

## How it Works
User inputs location and disaster type → AI generates survival guidance instantly.
